Reserved Words
--------------

The following words are reserved for the EL and should not be used as
identifiers:

`and` +
`or` +
`not` +
`eq` +
`ne` +
`lt` +
`gt` +
`le` +
`ge` +
`true` +
`false` +
`null` +
`instanceof` +
`empty` +
`div` +
`mod`
